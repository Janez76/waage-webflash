FilaPropus Waage WebFlash

Firmware + LittleFS direkt im Browser flashen (ESP32)

Schritt 1
  1. ESP32 per USB verbinden.
  2. Chrome oder Edge verwenden (WebSerial erforderlich).
  3. Unten auf "Flash" klicken.
Hinweis: Der Flash-Vorgang loescht vorhandene Daten auf dem ESP32.
Firmware installieren
Erster Start
  1. Nach dem Flashen startet der ESP32 neu.
  2. Mit dem WLAN "FilaPropus" verbinden (Config Portal).
  3. Dann im Browser http://filapropus.local oeffnen.
